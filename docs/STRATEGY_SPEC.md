# 512890红利低波策略规格

## 交易口径

- 信号日：T日收盘后。
- 执行日：下一交易日T+1开盘。
- 标的：只交易512890。
- 现金：唯一非风险资产。
- 本金默认：100,000元。

## 四信号

1. 相对强弱：`R = close_588000 / close_512890`。
2. 512890趋势结构：`MA5 / MA10 / MA20`。
3. 512890承接强度：`CLV = (close - low) / (high - low)`。
4. 科技拥挤/钝化：`588000 K_CLV`、MA5、市场上涨占比。

## 仓位状态

| 状态 | 仓位 |
|---|---:|
| S0 空仓等待 | 0% |
| S1 观察仓 | 20% |
| S2 初步确认 | 40% |
| S3 主仓持有 | 70% |
| S4 满仓持有 | 100% |

## 买入硬条件

- `S0→S1`：总分≥4、`R<1.70`、`CLV≥0.60`、未创5日新低、成交额不低于5日均额80%。
- `S1→S2`：总分≥6、`R<1.64`、512890站上MA5、承接增强、科技钝化或跌破MA5。
- `S2→S3`：总分≥8、`R<1.64`且R三日均线下行、512890站上MA10、连续3日强承接、科技转弱。
- `S3→S4`：总分≥10、`R<1.59`、512890站上MA20、MA5>MA10、科技弱、市场宽度修复。

## 卖出硬条件

- 卖出优先于买入。
- 异常下跌规则优先级最高。
- 观察仓亏损3%或R重新站上1.75清仓。
- 满仓状态出现两个减仓条件降至70%。
- 主仓状态出现三个减仓条件降至40%。
- 确认仓状态出现两个减仓条件降至20%。

## 执行规则

买入：

- T+1开盘价 ≤ T日收盘价×1.008：正常买入。
- 高开0.8%–1.5%：只买计划金额50%。
- 高开超过1.5%：暂缓，当日收盘后重新计算。

卖出：

- T+1开盘直接卖出，不设低开保护。

## 每日自动更新口径

- 自动运行时间：北京时间18:15，周一至周五。
- GitHub Actions工作流：`.github/workflows/daily-signal.yml`。
- 自动更新脚本：`scripts/update_daily_snapshot.py`。
- 自动数据文件：`data/live_prices.csv`。
- 最新信号快照：`data/latest_signal.json`。
- 自动组合状态：`data/auto_portfolio.json`。
- 历史信号记录：`data/signal_history.csv`。

每日更新流程：

1. 拉取512890与588000日线行情。
2. 合并历史数据并按日期去重。
3. 如果上一交易日已有pending order，则用今日开盘价模拟T+1执行。
4. 用今日收盘数据重新计算四信号与状态机。
5. 生成下一交易日开盘计划。
6. 写入JSON/CSV并自动提交。

自动组合仅用于信号可视化和复盘，不代表券商真实成交。真实买卖仍需用户在下一交易日开盘手动执行。
